# ML Mini Project

A comprehensive Machine Learning mini project showcasing the training, evaluation, and comparison of **multiple ML models** on a real-world dataset.

---

## 📋 Project Overview

This repository contains a complete end-to-end Machine Learning workflow. I explored, trained, and compared several classical and ensemble machine learning models to solve a **classification/regression** problem

**Objective**: Build, evaluate, and select the best performing model for the given problem while following standard ML practices.

---

## ✨ Features

- Exploratory Data Analysis (EDA)
- Data preprocessing and feature engineering
- Training of multiple ML algorithms
- Hyperparameter tuning
- Model evaluation and comparison
- Visualizations and performance metrics
- Pickle file of the best model
- Jupyter Notebook with full workflow

---

## 🛠 Technologies & Libraries

- **Python** 3.8+
- **Pandas** & **NumPy** – Data manipulation
- **Matplotlib** & **Seaborn** – Visualization
- **Scikit-learn** – ML models & preprocessing
- **Jupyter Notebook** – Development

---

## 📊 Models Trained

| Model                  | Type              | Used For          |
|------------------------|-------------------|-------------------|
| Logistic Regression    | Linear            | Baseline          |
| Decision Tree          | Tree-based        | Interpretability  |
| Random Forest          | Ensemble          | Performance       |
